# 📚 Website Belajar Online (Static HTML)

Website belajar online sudah selesai! 🎉  
Buka file **belajar-online.html** di browser untuk menikmati semua fiturnya.

---

## 🚀 Cara Menjalankan

1. Download atau clone repository ini  
2. Buka file:
belajar-online.html

3. Jalankan langsung di browser (Chrome / Edge / Firefox)

---

## ✨ Fitur Website

### 🏠 Beranda (Dashboard)
- Statistik belajar  
- Kursus yang sedang berjalan  
- Aktivitas terbaru  
- Kursus populer  

---

### 🎓 Semua Kursus
- Total **12 kursus** dari berbagai topik:
  - Pemrograman  
  - Data  
  - Desain  
  - Bisnis  
  - Bahasa  
- Filter kategori  
- Label:
  - 🟢 Gratis  
  - 🔵 Pro  
  - 🆕 Baru  
- Progress bar tiap kursus  

---

### ▶️ Video Pelajaran
- Player video interaktif  
- Playlist materi  
- Tombol:
  - Play / Pause  
  - Progress bar video  
  - Tandai selesai  

---

### 📝 Kuis & Latihan
- 3 topik kuis:
  - Python  
  - Excel  
  - HTML  
- Masing-masing **5 soal**  
- ⏱️ Timer 60 detik per soal  
- 📖 Penjelasan jawaban  
- 📊 Skor akhir  

---

### 📊 Progres Belajar
- Progress bar tiap kursus  
- Streak belajar mingguan  
- Statistik belajar lengkap  
- Sistem badge / pencapaian  

---

### 🏆 Sertifikat
- Menampilkan sertifikat yang sudah diraih  
- Tombol:
  - Unduh PDF  
  - Bagikan  
- Sertifikat terkunci sampai kursus selesai  

---

## 🛠️ Teknologi
